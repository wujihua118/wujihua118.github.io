    http-assert 错误捕获处理

    Evan 4/25/2021 httpassert

    node 搭建后台服务器，涉及到了服务端登录校验

    # 问题描述

    创建一个登录拦截中间件，在里面进行验证 token 的操作

    • 验证请求是否携带 token
    • 验证 token 中是否包含了用户 id
    • 到数据库中查找是否存在该用户

    因为这其中需要写很多的 if 判断，所以就用 http-assert 这个包来简化操作，将错误抛给错误处理中间件处理，然后返回给前端

    // 1.判断请求中是否携带 token
const token = String(req.headers.authorization || '').split(' ').pop()
assert(token, 401, '请先登录')
// if (!token) {
//   return res.status(401).send({message: '请先登录'})
// }

// 2.判断 token 中是否存在 id
const { id } = jwt.verify(token, app.get('secret'))
assert(id, 401, '请先登录')
// if (!id) {
//   return res.status(401).send({message: '请先登录'})
// }

// 3.根据 id 在数据库中查找是否存在该的用户
req.user = await User.findById(id)
assert(req.user, 401, '请先登录')
// if (!user) {
//   return res.status(401).send({message: '请先登录'})
// }


    const { username, password } = req.body
const user = await User.findOne({username}).select('+password')
assert(user, 422, '用户不存在')
// if (!user) {
//   return res.status(422).send({message: '用户不存在'})
// }
const isValid = require('bcrypt').compareSync(password, user.password)
assert(isValid, 422, '密码错误')
// if (!isValid) {
//   return res.status(422).send({message: '密码错误'})
// }
const token = jwt.sign({id: user._id}, app.get('secret'))
res.send({token})

    错误处理函数：

    app.use(async (err, req, res, next) => {
  res.status(err.statusCode || 500).send({
    message: err.message
  })
})

    前端响应拦截代码如下：

    http.interceptors.response.use(res => {
  return res
}, err => {
  if (err.response.data.message) {
    Vue.prototype.$message({
      type: 'error',
      message: err.response.data.message
    })
    if (err.response.status === 401) {
      router.push('/login')
    }
  }
  return Promise.reject(err)
})

    然后故意输入不存在的用户名发起登录请求时，服务器报了以下的错误：

    本应该是将错误统一交由错误处理函数来处理，然后发送给前端，前端做响应拦截，当输入数据库中不存在的用户时，前端 ui 应该提示用户不存在，然而没有，这并不符合我的预期

    # 解决方案

    通过上网查阅资料，解决方案如下：

    在异步函数外层套一层 try / catch ，捕获异步函数中抛出的错误，到 catch 中，将错误抛给错误处理函数，将错误信息返回给前端用户

    try {
  const token = String(req.headers.authorization || '').split(' ').pop()
  assert(token, 401, '请先登录')
  const { id } = jwt.verify(token, app.get('secret'))
  assert(id, 401, '请先登录')
  req.user = await User.findById(id)
  assert(req.user, 401, '请先登录')
} catch (error) {
  next(error)
}

    try {
  const { username, password } = req.body
  const user = await User.findOne({username}).select('+password')
  assert(user, 422, '用户不存在')
  const isValid = require('bcrypt').compareSync(password, user.password)
  assert(isValid, 422, '密码错误')
  const token = jwt.sign({id: user._id}, app.get('secret'))
  res.send({token})
} catch (error) {
  next(error)
}
