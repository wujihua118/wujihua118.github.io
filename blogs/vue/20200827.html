    element-ui按需引入报错Error:Cannot find module ‘babel-preset-es2015’

    vuePress-theme-reco Evan    2020 - 2021

    element-ui按需引入报错Error:Cannot find module ‘babel-preset-es2015’

    Evan 8/27/2020 vueelement-ui

    按 element-ui 官网的步骤进行按需引入，报错：Error: Cannot find module ‘babel-preset-es2015’

    # 安装 babel-plugin-component

    npm install babel-plugin-component -D

    # 修改 .babelrc 文件

    {
  "presets": [["es2015", { "modules": false }]],
  "plugins": [
    [
      "component",
      {
        "libraryName": "element-ui",
        "styleLibraryName": "theme-chalk"
      }
    ]
  ]
}

    报错：Error: Cannot find module ‘babel-preset-es2015’?

    # 问题描述

    项目是基于vue-cli 3.x的版本，并没有官网的.babelrc文件，只有babel.config.js文件，而且项目不需要引入完整的element-ui，只需要引入部分，按官网的步骤安装babel-plugin-component后在babel.config.js写入内容，报错

    # 解决办法

    .babelrc文件和babel.config.js文件都是配置文件，可以视为相同，之后需要使用npm安装

    npm i @babel/preset-env -D

    而且，也不再使用es2015，改成@babel/preset-env，两者是相同的，其作用都是编译es6语法，因此在babel.config.js中写成

    module.exports = {
  presets: [
    "@vue/cli-plugin-babel/preset",
    ["@babel/preset-env", { modules: false }],
  ],
  plugins: [
    [
      "component",
      {
        libraryName: "element-ui",
        styleLibraryName: "theme-chalk",
      },
    ],
  ],
};
